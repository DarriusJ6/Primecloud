# Security Policy

## Supported Versions
This repository is a portfolio and documentation project for a private lab environment. It is not a production software distribution with versioned security support.

## Responsible Disclosure
If you believe you have found a security issue in this repository (e.g., an accidentally committed secret), please report it responsibly:

- Do **not** publicly disclose the issue
- Contact: **jdarrius246@gmail.com**
- Include: file path(s), a description of the issue, and suggested remediation if available

## What NOT to Report
The following are out of scope for reporting:
- Home lab exposure risks outside of this repository
- Issues requiring access to my private network or systems
- Vulnerabilities in third-party software used in the lab (Proxmox, OPNsense, etc.)
- Findings based on guessing credentials or scanning my home IP space

## Commitment
I will respond to valid reports by removing sensitive material and rotating credentials if needed.
